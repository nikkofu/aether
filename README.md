# AetherCLI

> Agentic Developer Orchestration Runtime

AetherCLI is a multi-agent orchestration CLI designed to:

- Orchestrate Gemini CLI, Claude Code and future LLM CLIs
- Execute dynamic DAG-based tasks
- Enable policy-aware execution
- Provide auditable AI-assisted engineering

This is not an LLM wrapper.

This is the foundation of an Agentic Engineering OS.

---

## Why?

Single-loop AI coding is fragile.

We believe:

Decision → Policy → Execution → Observation → Memory

must be structured.
